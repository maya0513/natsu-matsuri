# ヨーヨー釣り

屋台いっぱいの大きな水槽に浮かぶ水風船（ヨーヨー）を、こよりで 1 つ釣る。**1 つ釣れたら終了**。

## 操作
- 十字キーでこよりを左右に動かし、Enter で釣る。/ 風船を直接クリック・タップでも釣れる。
- 外せる回数に上限あり（`triesLeft`）。

## 勝敗
- こより（またはタップした風船）の近くに生きた風船があれば釣れる（命中窓 `HIT_WINDOW.yoyo`）。
- **1 つ釣れたら即終了**。外し切る（triesLeft 0）と終了。

## 景品
- 釣れたら **水風船（yoyo-balloon）** を持ち帰る。

## 演出
- 水風船はデザイン（色）・大きさ・配置がすべて別々のランダム（赤・青・緑・黄・紫の 5 色）。
  水面でゆらゆら揺れ、釣ると上へ持ち上がって消える。こよりは決定時に水面へ沈む。

## 実装メモ
- 状態: `YoyoState { cursor, balloons[], triesLeft, caught, last? }`。`Balloon` は `kind`(色)・`size`・`x`・`baseY`(配置) を持つ。
- `isFinished`: `caught >= 1 || triesLeft === 0 || 全部消えた`。
- 風船スプライトは `minigame.png`（`yoyo-0`〜`yoyo-4`）。`size` で大きさを変える。
