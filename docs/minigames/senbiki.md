# 千本引き

横木から垂れた紐を 1 本選んで引く。先につながった景品が手繰り寄せられる。

## 操作
- 左右キーで紐を選び、Enter で引く。/ 紐を直接クリック・タップでも引ける。
- 引けるのは 1 回。

## 勝敗・確率
- 当たり判定は rng（`senbikiResult`）: 大当たり 8% / 当たり 42% / はずれ 50%。

## 景品
- はずれ以外で **景品（senbiki-prize＝だるま）** を持ち帰る。

## 演出
- 選択中の紐とその引換札が金色に光る。引くと当たり札が手元（手前・上）へ手繰り寄せられ、当たりなら景品が出る。
  何が起きたか分かるアニメーションで、文章通知はしない。

## 実装メモ
- 状態: `SenbikiState { count, cursor, picked?, result? }`。紐は **6 本**（少なめにして分かりやすく）。
- 紐は細い円柱、先にタグ（Box）。当たり景品は `held.png` の `senbiki-prize`。
