# もぐら叩き

盤面を斜め上から見下ろし、穴からゆっくり出るもぐらのおもちゃをハンマーで叩く反応ゲーム。

## 操作
- 十字キーでハンマーを左右に動かし、Enter で叩く。/ 出ているもぐらを直接クリック・タップでも叩ける。
- 叩ける回数に上限あり（`triesLeft`）。

## 勝敗
- ハンマー（またはタップ）の近くに「出ていて、まだ叩いていない」もぐらがいれば命中（命中窓 `MOGURA_WINDOW`）。引っ込んでいる・既に叩いたものは外れ。
- **2 匹以上**叩けば勝ち。ハンマー切れで終了。

## 景品
- 2 匹以上で **景品（mogura-prize）** を持ち帰る。

## 演出
- もぐらのおもちゃがゆっくりした周期で穴から出没する。叩くと**目が×の顔**に変わってしばらく固まり、そのあと穴に引っ込む。ハンマーは振り下ろされる。

## 実装メモ
- 状態: `MoguraState { cursor, moles[], triesLeft, hits, last? }`。`Mole` は `up`・`timer`・`stunned` を持つ。
- 周期は `MOGURA_UP`/`MOGURA_DOWN`（長め）。叩くと `stunned` をセット（出たまま判定外）→ 0 になると引っ込む（`stepMole`）。
- もぐらスプライトは `minigame.png`（`mole`/`mole-hit`）。ハンマーは `held.png` の `mogura-prize`。
