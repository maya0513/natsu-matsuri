# 金魚すくい

屋台いっぱいの大きな水槽を斜め上から覗き、水面を不規則に泳ぐ金魚をポイで狙ってすくう。

## 操作
- 十字キーでポイを縦横に動かし、Enter ですくう。/ 金魚を直接クリック・タップでもすくえる（離れていても狙える）。
- ポイは 2 回で破れる（`poiLeft`）。

## 勝敗
- ポイ（またはタップした金魚）の近くに生きた金魚がいればすくえる（命中窓 `HIT_WINDOW.kingyoX`/`kingyoY`）。
- 1 匹以上すくえば勝ち。ポイ切れか全部すくうと終了。

## 景品
- 1 匹以上で **金魚（goldfish＝水の入った袋）** を持ち帰る。何匹すくっても手持ちは袋 1 つ。袋はマップ上で手に持って歩く。

## 演出
- 金魚は 5 匹。和金（赤）・出目金（黒）・三色の複数デザインが、向きを少しずつ変えながら不規則に泳ぐ。
  すくうと水面から金魚が上へ抜けて消える。ポイは決定時に水面へ沈む。当たり/はずれの文章は出さない。

## 実装メモ
- 状態: `KingyoState { cursor, cursorY, fish[], poiLeft, caught, last? }`。
- 金魚は 2D 速度（`vx`/`vy`）＋向きの揺らぎで不規則に泳ぎ、水槽の縁で跳ね返る（`swimFish`）。
- 金魚スプライトは `minigame.png`（`fish-red`/`fish-black`/`fish-calico`）。小さめに表示。
- クリック取得は `mesh.userData.targetIndex` ＋ `commitMinigame(state, rng, target)`（target の x/y にカーソルをスナップ）。
