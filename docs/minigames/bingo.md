# ビンゴ

実物のビンゴ紙（5×5・中央フリー）を見ながら、ガラポンで玉を引いて数字をそろえる。

## 操作
- Enter、または盤面（紙/抽選器）をクリック・タップで玉を 1 つ引く。
- 玉は 1〜100（重複なし）。引いた数字が紙にあれば、そのマスに印が付く。

## 勝敗
- 行・列・対角のどれか 1 ライン（5 マス）がそろえばビンゴ。中央はフリーで最初から印。
- ビンゴになるか、玉を引ける回数（`BINGO_DRAWS` = 18）を使い切ると終了。

## 景品
- ビンゴで **景品（bingo-prize＝金メダル）** を持ち帰る。

## 演出
- ビンゴ紙は `CanvasTexture` で 3D の紙として描画（モーダルには数字を出さない）。1〜100 の数字、中央★、直近に出た玉を見出しに表示。一致するマスに赤丸の印。
  抽選器（ガラポン）が回り、出た玉が転がる。ビンゴ成立で玉が金色に光る。

## 実装メモ
- 状態: `BingoState { card[25], marked[25], drawn[], lastBall?, bingo }`。中央 index 12 は値 -1（フリー）。
- カード生成は rng（`initMinigame("bingo", rng)`）で 1〜100 のユニークな 24 マス。ライン判定は 5×5 の 12 ライン（行 5＋列 5＋対角 2）。
- 1 枚カードでも遊べるよう、玉はカードの未マーク数字に当たりやすく寄せている（`BINGO_HIT_BIAS`）。
