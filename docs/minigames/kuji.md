# くじ引き（おみくじ）

ほぼ正方形の木箱を上から覗き込み、詰まった三角折り札を 1 枚選んで引く。妖怪ウォッチのくじ引き画面が着想。

## 操作
- 十字キーでハイライトする札を移動、Enter で引く。/ 札を直接クリック・タップしても引ける。
- 引けるのは 1 回（引いたら結果へ）。

## 勝敗・確率
- 運勢は rng で決定（`fortuneFromRng`）: 大吉 10% / 中吉 15% / 小吉 20% / 吉 22% / 末吉 15% / 凶 13% / 大凶 5%。
- 吉系（大吉〜末吉）で当たり。

## 景品
- 吉系なら **お守り（omamori）** を持ち帰る。凶・大凶は景品なし。

## 演出
- 三角折り札を 3 列の格子に敷き詰める。選んだ札が箱から持ち上がり、おみくじの紙が開く。
  紙（2D の CanvasTexture）には朱色の帯と「御神籤」、運勢を大きく描く（凶系は藍、吉系は朱）。文章はモーダルに出さない。

## 実装メモ
- 状態: `KujiState { count, cursor, picked?, result? }`（札 9 枚＝3×3）。
- 札は横倒しの三角柱（`ConeGeometry(.., 3)`）、選択中は金色。
- カメラは正方形の箱を真上気味から覗き込むプリセット。
